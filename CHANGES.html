Version 3.6 (30.12.2014)

  • Dependency check for site deleting.
  • Map changed from Openlayers to Leaflet.

Version 3.3 (02.07.2014)

  • Checkbox to toggle between the classic Google Maps display and the current OSM/OpenLayers implementation.
  • New field "Country" at AS level to group the AS by ISO-3166 tld country codes. Used only for limiting generated output, see below.
  • Limit map content by country for generating overview sites.
    Examples: http://hamnetdb.net/lsp_map.cgi?country=at or http://hamnetdb.net/mapwindow.cgi?country=at.
  • Limit DNS generation (see Utilities) to a particular country if country parameter is given. This gives the possibility to create a whole country zone like at.ampr.org.
  • Bugfix: Avoid coordinate input exceeding valid degree values in the site entry form.
  • Bugfix: Entering a hostname into search field shows correct result.
  • Bugfix: Profile image is shown at 1920px window width or greater.

Version 3.2 (30.04.2014)

  • New map display based on OpenStreetmap and OpenLayers. Many thanks to Lucas, OE2LSP for design and implementation of this part :-)

Version 3.1 (23.03.2014)

  • Rights management: New option "Restrict write access to list of maintainers" for maintainers of AS, Subnet, Site, Host. Requires match of maintainer list to own callsign.

Version 3.0 (07.01.2014)

  • Bugfix: Link was not shown when first IP of subnet was empty.
